# 🔒 Cryptographic Hash Generator (Data Integrity Tool)

## Project Author & Affiliation
* **Author:** Harsh Chakravarti
* **Application Intent:** Applicant for the IIT Kanpur Bachelors in Cybersecurity Program 2026

## Overview
This is a Java-based security tool designed to demonstrate the core cybersecurity concept of **Data Integrity**. The program takes user input and passes it through one-way cryptographic hash functions to generate unique, fixed-size digital signatures. 

## Security Concepts Demonstrated
* **One-Way Encryption:** Utilizing standard Java security libraries to create irreversible hashes.
* **Data Integrity Verification:** Generating hashes that can be used to verify that data has not been altered or tampered with in transit or storage.
* **Algorithm Familiarity:** Implementing both **MD5** (legacy standard) and **SHA-256** (modern cryptographic standard) algorithms.

## Technical Details
* **Language:** Java (JDK 8+)
* **Libraries Used:** `java.security.MessageDigest`, `java.security.NoSuchAlgorithmException`
* **Core Logic:** Converts string byte arrays into formatted hexadecimal string representations.

## Motivation
This project was built to apply foundational Java programming and algorithmic logic to fundamental information security principles.
